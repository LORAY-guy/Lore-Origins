Both 2 and 3 icons format are taken into account by the engine automatically.
All you need to do is set the image to :
	- 300x150 resolution if it's a set of 2 icons
	- 450x150 resolution if its a set of 3 icons.

The expressions goes in the order "Losing", "Normal", and "Winning" (for the set of 3 icons)

THAT'S IT!

The way it works if by checking the width of the image
(the height doesn't actually matter, so i believe you are free to do whatever with that, BUT TRY TO RESPECT THE FORMAT AS MUCH AS POSSIBLE)
