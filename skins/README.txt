WELCOME TO THE SKINS FOLDER!!!!!!

Here you can add your own skins into the mod, as many as you want, and for whatever character you want to replace.

It's a little complicated, i'm sorry about that, but hey! It works.

Just add all the necessary assets into their respective folders (there's a README for every folder that will explain what you need to do)
and DON'T FORGET TO ADD THE CHARACTER IN THE "skinData.json" FILE.
(create it if its not there, the game will look for it every time you open the Skin Selector menu).

NOTE: The "skinData.json" that you will create will override the default skinData in "assets/data/skinData.json"

Enjoy!!!
